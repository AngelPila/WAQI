# Assets Directory

This directory should contain the following image assets for Expo:

## Required Assets

| File | Size | Description |
|------|------|-------------|
| `icon.png` | 1024x1024 | App icon for iOS and Android |
| `adaptive-icon.png` | 1024x1024 | Adaptive icon foreground for Android |
| `splash.png` | 1284x2778 | Splash screen image |
| `favicon.png` | 48x48 | Web favicon |

## Notes

- All images should be PNG format
- Icon should have no transparency
- Adaptive icon should have the foreground element only
- Splash screen background color is set to #22c55e in app.json

For more info, see: https://docs.expo.dev/guides/app-icons/
